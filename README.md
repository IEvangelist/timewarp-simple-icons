# timewarp-simple-icons

All the simple-icons wrapped as Blazor components.

## Simple-icons

## Commands used

```PowerShell
dotnet new sln
dotnet new razorclasslib -n timewarp-simple-icons
dotnet sln add .\Source\timewarp-simple-icons\timewarp-simple-icons.csp
```
